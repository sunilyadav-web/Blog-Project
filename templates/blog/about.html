About this Blog Project

Title - Sunil Code Blog
Description

This is a blogging platform project, which is built-in Python Django and Rest API. We have two main sections User and Blog Model.

What's in this Project and How it works
About User System
  • Register or Login
  • Activating user account via Email Verification link
  • Profile Section
  • User can edit Profile
  • can't add more than 150 character in bio (this is counting by Js and disable edit form if more than 150)
  • Change Eamil- if user change his email then our system verify that with email verification link (It show warning if not verify
  • Display user's followers ( how Many other users followed him)
  • Display all blog post that user only had posted
  • user can follow and unfollow users
  • user can see other users profile
About Blog
  • User can add blogs (if user account is not activate can't add blog) *Activating Account
  • Edit blogs
  • Delete blogs
  • users can Like or unlike blogs
  • user can seen all thier blogs list table
  • users can comment on the blog and they can edit thier comment and delete.
  • Edit comment functionality performed by Django Rest API
  • most liked blog will display in featured blog window
  • Displaying blogs in form of cards
  • What display in blog cards
    • Blog Image with hyper link of full blog detail
    • Blog publisher username with hyper link of his profile
    • Blog publish date and link of full blog detail
  • Provide froala editor to write blog conent
  • user can share blog by share button link
About other Features
  • Search blogs functionality
    • Search blog will display after matching searched keyword in blog title and blog content
    • search will give error if search query length greater than specified query length
    • if search keyword not match with any blogs content and title it will simply show you seggestions and warning
    • It will show all the blogs that match with your keyword
  • Pagination with next and previous button and you can also go on specific page by clicking on page number
